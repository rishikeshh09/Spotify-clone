About My Spotify Clone

Project Overview

This Spotify Clone is a personal project created to demonstrate web development skills and provide a functional music streaming experience. The application allows users to browse, search, and play music with a familiar interface inspired by Spotify.

The project was built with a focus on responsive design, user experience, and modern web technologies.

Features

The Spotify Clone includes the following features:

  • User authentication (login/registration)
  • Music playback with controls (play, pause, skip, volume)
  • Song search functionality
  • Responsive design for various screen sizes
  • Animated background with fixed positioning
  • Custom scrollable song list
  • Visual feedback for currently playing songs
  • Keyboard shortcuts for playback control
  • Space - play, side arrow keys - forward/backward 5 sec, up/down arrow keys - control volume

Technologies Used

This project was built using the following technologies:

HTML5 CSS3 JavaScript Node.js Express Font Awesome RESTful API
